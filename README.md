## 📌 Project Overview

This project is a Python-based Personal Finance Calculator developed as part of the Day 7 assignment.  

It demonstrates strong understanding of:

- Variables and Naming (PEP 8)
- Data Types (int, float, str, bool)
- Type Checking (`type`, `isinstance`)
- Type Casting
- f-strings Formatting
- Input Validation
- Python Truthiness
- Git Workflow
- AI-assisted evaluation

---

## 🚀 Features

### ✅ Part A – Core Financial Calculator
- Collects employee financial data
- Validates all inputs
- Calculates:
  - Monthly salary
  - Tax deduction
  - Net salary
  - Rent ratio
  - Savings
  - Disposable income
  - Annual projections
- Generates a formatted financial report

---

### ✅ Part B – Advanced Extension
- Indian number formatting (Lakhs/Crores system)
- Two-employee side-by-side comparison
- Financial Health Score (0–100) based on:
  - Rent ratio
  - Savings rate
  - Disposable income ratio

---

### ✅ Part C – Interview Ready Section
- Deep dive into Python data types
- Floating-point precision explanation
- Type analyzer function
- Debugging challenge (bug fixes implemented)

---

### ✅ Part D – AI-Augmented Task
- Generated Python type conversion matrix using AI
- Manually tested all conversions
- Identified AI inaccuracies
- Provided critical evaluation (accuracy analysis)
